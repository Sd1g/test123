# BPMN Визуализатор процессов

Веб-интерфейс для анализа бизнес-процессов и генерации BPMN диаграмм.

## Функции
- Анализ текстовых описаний бизнес-процессов
- Подсветка сущностей (AGENT, TASK, CONDITION)
- Генерация BPMN диаграмм через PlantUML
- Визуализация взаимодействий через тепловые карты
- Sankey диаграммы потоков между участниками
